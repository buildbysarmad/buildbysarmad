## Currently building

### [LogSage](https://logsage.dev) — Paste your logs. Know what broke and why.

> AI-powered log analysis for developers and small teams.

Upload or paste any log file. LogSage auto-detects the format, groups similar errors by pattern, and surfaces root causes — so you can fix things faster instead of reading noise.

**What it does**
- Auto-detects Serilog, NLog, Log4Net, standard, and plain text formats
- Groups similar errors by pattern automatically
- Surfaces root causes — not just what complained, but what actually broke
- No account needed to start — paste logs, get answers

**How it's built**
- Backend: .NET 9 Minimal API + PostgreSQL + EF Core 9
- Frontend: Next.js 15 + Tailwind CSS + shadcn/ui
- Auth: JWT + refresh token rotation
- Infra: Docker + GitHub Actions + Railway + Netlify
